Lógica de programação

Lógica é a ordenação do pensamento, por que nossa razão pode funcionar desordenadamente. Por isso precisamos de uma ciência das formas de pensamentos.
Exemplo: Todo mamífero é um animal. Todo cavalo é mamífero. Portanto, todo cavalo é um animal.
O raciocício lógico pode ser expressado de várias formas, nas várias linguas existentes no planeta e todos estarem corretos; a lógica de programação existe para representar com fidelidade este raciocínio, através dos Algorítmos.

Eu e você recebemos a instrução: “faça um bolo para o seu namorado.” Você pode pensar da seguinte forma:
passo 1: Saber qual sabor ele gosta
passo 2: Comprar os ingredientes
Passo 3: Fazer o bolo
Já eu poderia pensar:
passo 1: aprender a fazer bolo
passo 2: descobrir qual sabor ele gosta
passo 3: Comprar os ingredientes
passo 4: ingredientes estão acima do orçamento
passo 5: cancelar o bolo

Algorítmos são simplesmente uma sequência de passos para atingir um objetivo. Ou seja o raciocínio lógico organizado em passos claros, com intúito, início, tempo de execução e finalização bem definidos, garantem que sempre que for executado, trará os mesmos resultados.
Se Matheus estuda Banco de Dados, então Rodolpho estuda Cobol. Se Camilla estuda Design, então José estuda Medicina. ORA, Matheus estuda Banco de Dados ou Camilla estuda Design. Logo, podemos afirmar que?
A- Rodolpho estuda Cobol ou José estuda Medicina
B- Se Matheus não estuda Banco de Dados, então José não estuda Medicina
C- Carla estuda Design e Rodolpho estuda Cobol
D- Rodolpho estuda Cobol ou Carla não estuda Design

Sua tarefa é raciocinar a lógica para o desafio acima, em breve postarei a resposta!
